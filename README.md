# To-Do List Application

## Description
This repository contains a simple to-do list application built with React, TypeScript, Vite, Tailwind CSS, Redux, and Redux Toolkit.

## Getting Started

### Installation
```bash
$ npm install

Development Server
$ npm start

Simulating Server Environment
$ npm run app-server

Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- Redux
- Redux Toolkit
